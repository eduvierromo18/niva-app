# Design System

La interfaz debe sentirse limpia, moderna y alineada con un SaaS premium.

## Principios visuales

- Usar mucho espacio en blanco.
- Usar bordes redondeados.
- Usar sombras suaves.
- Usar verde como color principal.
- Usar rojo solo para gastos, deudas y alertas negativas.
- Usar azul o tonos neutros para informacion.
- Evitar saturar tarjetas con colores fuertes.
- Priorizar claridad sobre densidad.
