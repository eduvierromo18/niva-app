# Meridian Implementation

## Status

- Version: 1.0.0
- Status: Implemented
- Source: Niva Meridian Identity v1.0
- Product surface authority: Niva Design Manual v4.0

## Implemented Assets

- Positive mark: graphite uprights `#111827`, ultramarine crossing `#27409A`.
- Dark-ground mark: knockout uprights `#FCFCFD`, lifted ultramarine crossing `#6E8AD6`.
- Wordmark: Archivo Light at `-0.03em`.
- Product typography: locally hosted Inter, Manrope and IBM Plex Mono.
- Install icon: sanctioned dark-ground mark in the approved rounded threshold container.
- Web app manifest: standalone display, Spanish locale and Niva metadata.

## Usage Rules

- Ultramarine is exclusive to the mark and never becomes a product accent.
- Product actions use Design Manual evergreen.
- The mark geometry must not be redrawn, rounded, rotated or recolored.
- The wordmark is not ordinary heading text.
- Positive and dark-ground variants are selected by their surrounding ground.

## Source Assets

The local WOFF2 files were extracted from the approved UI v5 compliance artifact so production does not depend on remote Google Fonts.
