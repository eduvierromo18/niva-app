# Colors

## Purpose

Define brand color direction before token-level implementation.

## Status

Draft

## Direction

Niva uses a calm, premium financial palette:

- Green as the primary product accent.
- Red for negative financial states.
- Amber for caution.
- Blue or neutral tones for information.
- Soft neutral backgrounds for workspace calm.
