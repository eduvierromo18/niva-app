# Iconography

## Purpose

Define iconography direction for Niva.

## Status

Draft

## Direction

Icons should support recognition and scanning. They should not carry meaning that is unavailable in text or layout.

## Current Rule

Use familiar, consistent symbols for actions, navigation, financial states, and feedback.
