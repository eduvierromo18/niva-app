# Brand Architecture

Niva's brand is not a collection of visual assets.

It is a decision system.

Every design, product and engineering decision should be traceable back to the brand architecture.

## Decision Flow

Before making any product or design decision, ask:

1. Does it support the Purpose?
2. Does it respect the Values?
3. Does it match the Personality?
4. Does it use the right Voice?
5. Does it use Niva Vocabulary?
6. Does it follow the Visual Language?
7. Does it improve the Product Experience?

If any answer is no, the decision should be reconsidered.

---

## Product Implications

1. No screen should be designed only from aesthetic preference.
2. No copy should be written outside the approved vocabulary.
3. No component should introduce a new visual language.
4. No feature should contradict the brand personality.
5. Product, design and engineering decisions must share the same source of truth.
