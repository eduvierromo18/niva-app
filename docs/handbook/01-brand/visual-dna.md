# Visual DNA

## Source

This chapter consolidates existing direction from `docs/DESIGN_SYSTEM.md`, `docs/UX_BIBLE.md`, and `docs/ui-architecture.md`.

## Direction

Niva should feel clean, modern, calm, and aligned with a premium SaaS product.

## Visual Principles

- Use generous white space.
- Prioritize clarity over density.
- Use soft backgrounds and white surfaces.
- Use subtle borders and shadows.
- Use green as the primary accent.
- Use red only for expenses, debt, and negative alerts.
- Use blue or neutral tones for information.
- Avoid saturated cards and decorative excess.

## Product Feeling

The interface should feel:

- Clear.
- Useful.
- Premium.
- Trustworthy.
- Calm.
