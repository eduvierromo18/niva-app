# Coding Standards

## Purpose

Document repository standards that affect future implementation.

## Status

Draft

## Commit Convention

Use Conventional Commits:

- `feat:`
- `fix:`
- `refactor:`
- `docs:`
- `style:`
- `test:`
- `chore:`

## Verification

Before product changes are merged, run:

```bash
npm run lint
npm run build
```
