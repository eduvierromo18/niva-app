# Frontend

## Purpose

Define frontend documentation rules.

## Status

Draft

## Current Direction

Niva uses Next.js, React, TypeScript, Tailwind CSS, and reusable components.

## Naming Rule

Niva is the product name. Internal `Aurora*` component names remain valid until a dedicated technical rename sprint.

## Documentation Rule

Product specs may reference existing components, but documentation-only sprints must not modify components, screens, hooks, styles, or UI behavior.
