# Activity

## Overview

Activity is the user's financial timeline.

## Purpose

Help the user review what changed, where money moved, and which records need attention.

## User Goals

- Scan recent movements.
- Filter by period, account, category, and type.
- Understand individual transaction context.
- Edit or review records safely.

## Primary Question

What happened with my money?

## Information Hierarchy

1. Date and period context.
2. Movement list or table.
3. Filters.
4. Totals and grouped signals.
5. Empty or review states.

## Components

- Filters.
- Data table.
- Transaction item.
- Currency amount.
- Empty state.
- Movement dialog.

## Interactions

- Filter records.
- Open movement detail.
- Create or edit movement through existing flows.

## States

- Populated timeline.
- Empty filtered results.
- Empty account history.
- Loading state.

## Future Ideas

- Premium timeline grouping.
- Smart review flags.
- CSV import and export.
