# Goals

## Overview

Goals help the user track progress toward savings objectives.

## Purpose

Make long-term financial progress visible and motivating without exaggeration.

## User Goals

- See goal progress.
- Understand remaining amount.
- Track target dates.
- Decide what to contribute next.

## Primary Question

Am I moving toward what I want?

## Information Hierarchy

1. Goal name and current progress.
2. Saved amount and target amount.
3. Remaining amount.
4. Target date or pace.

## Components

- GoalProgress.
- AuroraGoalCard.
- CurrencyAmount.
- EmptyState.

## Interactions

- Create goal.
- Review goal detail.
- Update progress through future flows.

## States

- On track.
- Behind.
- Completed.
- Empty goals.

## Future Ideas

- Simple projection.
- Contribution suggestions.
