# Values

## Clarity

The user should understand their financial state without unnecessary effort.

## Trust

Financial information must feel precise, stable, and honest.

## Calm

Niva should reduce pressure. It should not create urgency without cause.

## Usefulness

Every screen, message, and component should support a real user task.

## Continuity

Product decisions should compound. The Handbook protects consistency across sprints.
