# Layouts

## Purpose

Define layout behavior for Niva product screens.

## Status

Alpha

## Scope

Applies to app shell, page scaffolds, filter sidebars, dashboards, tables, and module pages.

## Rules

- Use a minimal top bar and predictable navigation.
- Use soft workspace backgrounds with clear content surfaces.
- Use fixed filter sidebars for analysis and records when useful.
- Keep repeated financial views scannable and stable.
