# Hierarchy

## Purpose

Define how Niva orders information on screen.

## Status

Draft

## Scope

Applies to page titles, metrics, cards, tables, charts, filters, empty states, and financial summaries.

## Rules

- Start with the primary financial question.
- Make totals and deltas easy to scan.
- Keep secondary explanation close to the data it explains.
- Avoid competing visual emphasis inside one surface.
