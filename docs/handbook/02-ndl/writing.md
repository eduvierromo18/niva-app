# Writing

## Purpose

Define interface writing rules for Niva.

## Status

Alpha

## Scope

Applies to buttons, empty states, labels, helper text, alerts, insights, onboarding, and product specs.

## Rules

- Be clear and direct.
- Be calm around financial stress.
- Be precise with numbers and states.
- Avoid exaggerated motivation.
- Use copy that helps the next action.
