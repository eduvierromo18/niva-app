# Patterns

## Purpose

Define reusable experience patterns for Niva.

## Status

Draft

## Scope

Applies to financial summaries, quick actions, filters, insight cards, progress indicators, lists, and module navigation.

## Current Patterns

- Summary first, detail second.
- Primary question before secondary metrics.
- Action near context.
- Empty states that explain the next useful step.
- Tables for comparison and repeated review.
