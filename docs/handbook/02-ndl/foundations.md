# Foundations

## Purpose

Define the base experience principles for Niva.

## Status

Alpha

## Scope

Applies to every product surface, especially dashboard, accounts, activity, budgets, goals, analytics, liabilities, and scheduled transactions.

## Principles

- The interface must feel clear, premium, and useful.
- Every screen needs visible hierarchy.
- Visual decisions must support real tasks.
- The user should understand their financial state without unnecessary effort.
