# Spacing

## Purpose

Define how space supports calm and clarity.

## Status

Draft

## Scope

Applies to page structure, cards, forms, tables, toolbars, charts, and empty states.

## Rules

- Use spacing to separate tasks, not to decorate.
- Keep dense financial information readable.
- Preserve enough whitespace for calm scanning.
- Avoid visual crowding inside cards.
