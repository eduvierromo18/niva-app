# Elevation

## Purpose

Define depth and shadow direction.

## Status

Draft

## Rules

- Use subtle shadows.
- Prefer borders for structure.
- Reserve stronger elevation for overlays and active focus.
- Avoid decorative depth that competes with financial data.
