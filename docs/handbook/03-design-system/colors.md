# Colors

## Purpose

Define interface color rules.

## Status

Alpha

## Rules

- Primary accent: deep green.
- Positive state: green.
- Negative state: red.
- Warning state: amber.
- Information state: blue or neutral.
- Backgrounds: soft cool neutrals.
- Surfaces: white in light mode and dark zinc surfaces in dark mode.
