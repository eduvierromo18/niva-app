# Spacing

## Purpose

Define interface spacing direction.

## Status

Draft

## Rules

- Use generous space around major sections.
- Use tighter, predictable spacing inside repeated financial rows.
- Keep form controls grouped by task.
- Avoid nested card-heavy layouts.
