# Charts

## Purpose

Define chart usage direction for Niva.

## Status

Draft

## Rules

- Use charts to explain financial direction, not decorate.
- Pair visual data with readable totals or labels.
- Avoid large charts when a simple summary is more useful.
- Make comparisons easy to scan.
