# Typography

## Purpose

Define interface typography rules.

## Status

Alpha

## Rules

- Use clear system sans typography.
- Use strong weight for key metrics.
- Keep labels short.
- Preserve readable line length.
- Avoid decorative typography in operational surfaces.
