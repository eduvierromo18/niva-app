# Niva Product Vision

Niva es un Centro de Control Financiero Personal.

Tagline provisional: Tu espacio financiero.

Debe ayudar a responder de forma clara y rapida:

1. ¿Cuanto dinero tengo?
2. ¿Como voy este mes?
3. ¿En que se esta yendo mi dinero?

## Inspiracion

- Wallet por rapidez.
- Copilot y Monarch por insights.
- Linear y Vercel por limpieza visual.

Estas referencias sirven como direccion de producto y calidad visual, pero no se deben copiar interfaces exactas.

Nota tecnica: el naming interno `Aurora*` sigue vigente hasta un sprint tecnico de renombre.
