# UX Bible

## Proposito

Definir los principios de experiencia que deben guiar cada decision de producto.

## Principios base

- La interfaz debe sentirse clara, premium y util.
- Cada pantalla debe tener una jerarquia evidente.
- Las decisiones visuales deben apoyar tareas reales.
- El usuario debe entender su estado financiero sin esfuerzo innecesario.

## Uso

Este documento se actualizara cuando una decision de UX sea validada y reutilizable.
