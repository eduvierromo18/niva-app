# Motion System

## Proposito

Definir los principios de movimiento, transiciones y microinteracciones del producto.

## Principios base

- El movimiento debe orientar, no decorar.
- Las transiciones deben reforzar continuidad entre estados.
- La animacion debe ser breve, sobria y consistente.
- La experiencia debe seguir siendo usable sin depender del movimiento.

## Uso

Cada Product Spec que incluya movimiento debera documentar reglas concretas antes de implementarse.
